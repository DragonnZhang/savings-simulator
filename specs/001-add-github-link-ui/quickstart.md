# Quickstart: Savings Simulator

## Prerequisites
- Node.js 20+
- npm

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   # Run only the web app
   npm run dev
   ```

3. Open `http://localhost:3000`

## Verification
- Check the top-right corner for the GitHub link.
- Hover over the link to see the "premium" effect (glass/glow).
- Click the link to ensure it opens the repository in a new tab.
